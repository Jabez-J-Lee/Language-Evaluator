Scanner is written in C++ on VS Code.
Create both "test_input.txt" and "test_output.txt" if not provided.
Put all regex expressions you wish to test in "test_input.txt".
Make sure "test_output.txt" is empty prior to execution.
In the Terminal Type:
  g++ main.cpp -o main
Once that has completed Run the command with the text files with the line:
  ./main test_input.txt test_output.txt
Open "test_output.txt" to see results.
